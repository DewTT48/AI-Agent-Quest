Fill the Missing Canvas

ข้อมูลที่ให้แล้ว

Input: ปัญหาการทำงาน, กลุ่มพนักงาน, เป้าหมายการพัฒนา Process: วิเคราะห์ปัญหา → หา Skill Gap → เสนอหัวข้ออบรม Output: ตาราง Training Need

ให้ผู้เรียนเติม

Role: __________ Goal: __________ Human Review: __________
Suggested: Role = Learning & Development Advisor / Goal = วิเคราะห์ปัญหาและแปลงเป็น Training Need / Human Review = HR ตรวจว่าหัวข้ออบรมสอดคล้องกับบริบทองค์กรก่อนใช้
Facilitator Note: ให้คิดเป็นกลุ่ม 3–5 นาที แล้วสุ่มถาม 2–3 กลุ่มก่อน reveal

Suggested Answer

Role: Learning & Development Advisor Goal: ช่วย HR วิเคราะห์ปัญหาการทำงานและแปลงเป็น Training Need Input: 1. ปัญหาการทำงาน 2. กลุ่มพนักงาน 3. เป้าหมายการพัฒนา Process: 1. วิเคราะห์ปัญหาที่เกิดขึ้น 2. ระบุ Skill Gap ที่เกี่ยวข้อง 3. จัดกลุ่มความต้องการพัฒนา 4. เสนอหัวข้ออบรมที่เหมาะสม Human Review: ให้ HR ตรวจว่าหัวข้ออบรมสอดคล้องกับบริบทองค์กรก่อนนำไปใช้ Output: ตาราง Training Need Analysis
คำตอบไม่จำเป็นต้องเหมือนกันทุกกลุ่ม แต่ต้องครบ 6 ช่องและใช้ทำงานได้จริง
